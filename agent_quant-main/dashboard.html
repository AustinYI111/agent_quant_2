📚策略说明
📈
趋势跟踪策略 (Trend-only)
基于双均线交叉信号（短期MA(5) / 长期MA(20)）。当短期均线上穿长期均线时买入， 下穿时卖出。捕捉中长期价格动量趋势，适合趋势明显的行情。
信号 = sign(MA_short - MA_long)
↕️
均值回归策略 (MeanRev-only)
基于布林带（Bollinger Bands）的反转策略。价格跌破下轨（mu - 1.2*sigma）时买入， 回归均线时卖出。适合震荡行情，利用价格均值回复特性。
信号 = 1 if P < mu - 1.2*sigma else (-1 if P > MA)
🔀
融合策略 (Fusion no-ML)
加权融合趋势信号与均值回归信号（默认权重 0.6 : 0.4）。 通过 CoordinatorAgent 综合两种策略的置信度分数，当综合信号强度超过阈值时触发交易。
Score = 0.6*trend + 0.4*meanrev; 买入 if Score > theta
🧠
ML 融合策略 (Fusion +ML-veto)
在融合策略基础上增加 XGBoost 机器学习否决机制。当 ML 模型预测下跌概率 超过阈值（默认 0.55）时，否决买入信号，减少假阳性交易。
最终信号 = Fusion信号 AND (ML_prob < 0.55)
